## Dependencies:

* `PHP`
* `CURL`
* `OPENSSH-SERVER`
* `GIT`

## Installation :

* `apt update`
* `apt install git curl php openssh-server -y`
* `git clone https://github.com/OB1T0-17/FISH-X.git`
* `cd FISH-X`
* `bash fish-x.sh`
